# Integrated Communication Automation (Linux + Python)

This project automates **Email (via Gmail SMTP)**, **Voice Calls**, and **WhatsApp messages** from a single Python script on a **Linux system**.  
It was built as part of a practical learning journey under the mentorship of **Vimal Daga (LinuxWorld)**.

## Features
- Send Emails using Gmail SMTP (with App Passwords).
- Make Voice Calls using Twilio API.
- Send WhatsApp messages via Twilio WhatsApp API.
- Built and tested on RHEL 9 (Linux) environment.

## Setup

### Install Dependencies
```bash
pip install twilio
```

### Run the Project
```bash
python3 main.py
```
